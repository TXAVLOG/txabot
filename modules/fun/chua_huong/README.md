# Module: chua_huong ⛩️

Lệnh đi chùa thắp hương cầu may mắn cho bot Zalo.

## Cách dùng

```
.chua                     → Đi chùa cầu nguyện random
.chua cầu thi đỗ đại học  → Đi chùa với lời cầu tùy chỉnh
```

## Assets
- `assets/lotus.png`   — ảnh hoa sen (placeholder, thay bằng ảnh thật)
- `assets/incense.png` — ảnh thắp hương (placeholder)
- `assets/temple.png`  — ảnh chùa (placeholder)

> **Ghi chú**: Thay 3 file ảnh trên bằng ảnh thật để đẹp hơn.
> Kích thước khuyến nghị: 400x400px, format JPG/PNG.
